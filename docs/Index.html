Architektura Pipeline

Moduł 01
Ingestion
Automatyczne pobieranie cen GPW i nagłówków z polskich mediów finansowych co tydzień.

Moduł 02
NLP
Tłumaczenie PL→EN i analiza sentymentu modelem FinBERT. Agregacja do dziennego score per spółka.

Ekonometria

statsmodels Granger ARIMAX ADF test

Metodologia

✦ Co projekt robi
  • Pobiera ceny 10 spółek WIG20 przez yfinance (dane dzienne OHLCV)
  • Scraping newsów: Bankier.pl RSS + Google News RSS per spółka
  • Tłumaczenie PL→EN i analiza sentymentu FinBERT (score [-1, +1])
  • Agregacja dziennego sentymentu + lagi 1–6 dni sesyjnych
  • Test ADF (stacjonarność) + test Grangera dla lagów 1–10
  • Grid search rzędu ARIMA + porównanie ARIMA vs ARIMAX na test secie (20%)
  • Automatyczne odświeżanie danych co tydzień (Windows Task Scheduler)
⚠ Ograniczenia i plany
  • Mała próba (~57 sesji) — wyniki wymagają walidacji na dłuższym szeregu
  • FinBERT wytrenowany na tekstach angielskich — tłumaczenie PL→EN wprowadza szum
  • Pojedynczy split train/test zamiast rolling window cross-validation
  • Planowane: porównanie FinBERT vs HerBERT (polski BERT od Allegro)
  • Planowane: rozszerzenie historii do 2 lat gdy Task Scheduler zbierze dane
  • Planowane: interaktywny dashboard Plotly/Dash
